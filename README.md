# Bad Apple
Bash script to kill _unnecessary_ services running on OS X.

# Instructions

1. Make sure the file can be executed.

   `$ chmod +x osx_kill.sh`

2. It is recommended that you run the script with root privilleges.

   `$ sudo ./osx_kill.sh`
